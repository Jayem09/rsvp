# Baby Boss RSVP 🍼💼

A premium, high-end digital invitation with a "Cute Executive" personality. Designed for the Boss's big meeting!

## Features
- **Cute Executive Theme**: Gold shimmers, glassmorphism, and baby-themed professional icons.
- **Interactive Props**: Animated typewriter greeting and magnetic avatar hover effects.
- **Executive Dashboard**: A secret admin portal to manage the "Sealed Deals" (RSVPs).
- **Persistent Storage**: Uses SQLite for attendance tracking.

## Tech Stack
- **Frontend**: HTML5, Vanilla CSS3 (Custom animations), Vanilla Javascript.
- **Backend**: Node.js, Express, SQLite3.

## Local Setup
1. Clone the repository.
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the meeting:
   ```bash
   npm start
   ```
4. Access the boardroom: `http://localhost:3000`

## Admin Dashboard
To access the secret guest list, click the **Executive Seal** (the number "1" badge) **5 times** rapidly.
